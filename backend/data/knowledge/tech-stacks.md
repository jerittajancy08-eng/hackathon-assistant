# Tech Stack Selection

Category: Tech Stacks
Source: internal-hackbot-guide

## React

React is a strong frontend choice for interactive dashboards, chat interfaces, admin panels, and hackathon demos. Use it with Vite for fast setup and fast iteration.

## Node.js

Node.js with Express is useful for REST APIs, authentication, integrations, webhooks, and connecting frontend apps to AI APIs or databases.

## FastAPI

FastAPI is a good backend choice for Python-heavy projects, AI services, data processing, and ML model APIs. It provides automatic API docs and works well with Python AI libraries.

## Firebase

Firebase is useful for fast authentication, realtime databases, hosting, and serverless functions. It is a strong hackathon choice when speed matters more than custom infrastructure.

## MongoDB

MongoDB is useful for flexible JSON-like data, chat histories, user profiles, submissions, and rapidly changing schemas.

## PostgreSQL

PostgreSQL is best when data relationships, reporting, transactions, and reliability matter. Use it for structured apps, analytics, and production-ready relational data.

## Supabase

Supabase gives PostgreSQL, authentication, storage, edge functions, and realtime features. It is a strong alternative to Firebase when teams want SQL plus a fast backend platform.

Recommended hackathon stacks:

- AI assistant: React, Node.js or FastAPI, Groq/OpenAI-compatible API, MongoDB or PostgreSQL, vector store.
- Web3 app: React, Node.js, wallet integration, Solidity, Hardhat, Ethers or Viem, IPFS if needed.
- Healthcare MVP: React, FastAPI, PostgreSQL, secure auth, careful disclaimers, audit-friendly logs.
- FinTech MVP: React, Node.js, PostgreSQL, strong validation, role-based access, clear risk messaging.
- Education app: React, Firebase or Supabase, AI API, simple analytics dashboard.
