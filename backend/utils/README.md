# Hackathon Assistant 🚀

AI-powered hackathon discovery platform.

## Tech Stack

- React + Vite
- Node.js + Express
- MongoDB
- Ollama AI
- Tailwind CSS

## Features

- AI chatbot
- Hackathon recommendations
- Beginner-friendly filters
- AI/Web3 hackathon search
- MongoDB storage
- Chat sessions

## Run Frontend

```bash
cd frontend
npm install
npm run dev
